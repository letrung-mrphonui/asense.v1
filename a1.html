🔗 Apps Script Web App URL
Đây là URL duy nhất cần điền. Dashboard sẽ gọi URL này để lấy toàn bộ data.
Xem hướng dẫn deploy trong file asense_apps_script.js
⏱ Tự Động Refresh
📋 Hướng Dẫn Deploy Apps Script
1. Mở Google Sheet → Extensions → Apps Script
2. Xoá code cũ → Paste toàn bộ nội dung file asense_apps_script.js
3. Bấm Deploy → New Deployment → chọn type Web App
4. Execute as: Me  |  Who has access: Anyone
5. Bấm Deploy → Copy URL → Paste vào ô trên
6. Bấm Lưu & Kết Nối bên dưới
⚠ Mỗi lần sửa code Apps Script phải Deploy lại (New Deployment)
